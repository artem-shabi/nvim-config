# Neovim конфиг (NvChad)

Персональная конфигурация Neovim на базе [NvChad](https://nvchad.com/) v2.5.

---

## Тема

**Kanagawa** — тёмная тема в японском стиле.

---

## Языки и инструменты

| Язык | LSP | Форматтер |
|------|-----|-----------|
| Go | `gopls` | `gofumpt`, `goimports`, `goreturns` |
| HTML | `html` | — |
| CSS | `cssls` | — |
| Lua | — | `stylua` |

Синтаксическая подсветка (Treesitter): `vim`, `lua`, `vimdoc`, `html`, `css`, `go`.

## Клавиши

**Leader**: `Space`

### Общие

| Клавиши | Режим | Действие |
|---------|-------|---------|
| `;` | Normal | Войти в командный режим (`:`) |
| `jk` | Insert | Escape |
| `<C-s>` | Insert | Сохранить файл |

### Перемещение строк

| Клавиши | Режим | Действие |
|---------|-------|---------|
| `<A-j>` | Normal | Переместить строку вниз |
| `<A-k>` | Normal | Переместить строку вверх |
| `<A-j>` | Visual | Переместить выделение вниз |
| `<A-k>` | Visual | Переместить выделение вверх |

### Сессии

| Клавиши | Действие |
|---------|---------|
| `<leader>ss` | Восстановить сессию текущей директории |
| `<leader>sl` | Восстановить последнюю сессию |
| `<leader>sd` | Не сохранять текущую сессию |

Все стандартные клавиши NvChad также доступны.

## Структура

```
~/.config/nvim/
├── init.lua                  # Точка входа, bootstrap lazy.nvim
└── lua/
    ├── chadrc.lua            # Тема и настройки NvChad
    ├── options.lua           # Опции Neovim
    ├── mappings.lua          # Кастомные клавиши
    ├── autocmds.lua          # Автокоманды
    ├── plugins/
    │   └── init.lua          # Список плагинов
    └── configs/
        ├── lazy.lua          # Настройки менеджера плагинов
        ├── lspconfig.lua     # Конфигурация LSP
        └── conform.lua       # Конфигурация форматтеров
```
